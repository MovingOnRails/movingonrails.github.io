Project Aggregator

Project aggregator es un pequeño sitio de blogging/comercio electrónico desarrollado usando JAVA y Spring Boot. Cuenta con una forma de iniciar sesión como usuario administrador que puede publicar y editar productos o elementos del blog que aparecen en la página de productos de la aplicación. Llamamos a la plataforma Marinabi como una combinación de "Mariposa" y "Nabi" (mariposa en koreano).

Tecnologías:

JAVA, Spring boot MVC, Spring Security, MySQL, docker, maven , html, css, bootstrap, prism.js, pebble templating engine, Javascript, JPA, Hibernate
