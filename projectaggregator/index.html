Project Aggregator

Project aggregator is a small blog/e-commerce site developed using JAVA and Spring boot. It features a way to log in as an admin user that can publish and edit blog/product items, which appear on the products page of the application. We named the platform Marinabi as a combination of "Mariposa" (butterfly in spanish) and "Nabi" (butterfly in korean).

Technologies:

JAVA, Spring boot MVC, Spring Security, MySQL, docker, maven , html, css, bootstrap, prism.js, pebble templating engine, Javascript, JPA, Hibernate
